This plugin adds tabs functionality to the application.

For detailed documentation see https://github.com/spatie/vue-tabs-component .

Styling is at components/c-tabs.scss

```
<tabs :options="{ useUrlFragment: false }">
  <tab name="First tab">
    First tab content
  </tab>
  <tab name="Second tab">
    Second tab content
  </tab>
  <tab name="Third tab">
    Third tab content
  </tab>
</tabs>
```
